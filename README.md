# 📉 IBM Telco Customer Churn Analysis

Interactive Power BI dashboard analyzing 
customer churn patterns for a telecom company.

## 📸 Dashboard Preview
![Dashboard](Screenshot%202026-03-30%20112429.png)

## 📊 Key Metrics
- Total Customers: 7,043
- Churned Customers: 1,869
- Churn Rate: 26.54%
- Monthly Revenue: $456.12K

## 🔍 Key Insights
- Month-to-month customers churn the most (88%)
- Fiber optic users have highest churn (69.4%)
- Electronic check users churn more than others
- Churned customers pay $13 more per month
- Gender has no significant impact on churn
